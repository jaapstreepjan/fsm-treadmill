◆ lockStatus_t

enum lockStatus_t

Finite State Machine Library.

Implementation of the finite statemachine for a turnstile as defined by Robert C. Martin in UML Tutorial: Finite State Machines Code uses a pre-defined state handler (fsm_functions) adapted from code created by Hugo Arends, HAN by John van den Hooven

Implementation is based on Figure 2 in the document

This version of the code is a monolitic implementation All functions and subsystem code is implemented in main.c Development Console Library
